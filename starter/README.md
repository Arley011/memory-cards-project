# Memory Cards — Starter Project

This is **your base project**. You will build the Memory Cards app by expanding this starter step by step over the course.

## How to get started

1. Make sure Flutter is installed — see [../docs/setup.md](../docs/setup.md)
2. Open the `starter/` folder in VS Code (`File → Open Folder`)
3. Run `flutter pub get` in the terminal
4. Connect a device or start an emulator
5. Run `flutter run`

You should see an empty feed screen with a purple "+" button. That's your starting point!

## Project structure

```
lib/
├── main.dart               ← App entry point and theme
├── models/
│   └── entry.dart          ← The Entry data class (title, text, date, tags, photo)
├── data/
│   └── sample_entries.dart ← Hardcoded example data for Stage 1
├── screens/
│   └── home_screen.dart    ← The main feed screen (you'll add more screens here)
├── widgets/
│   └── entry_card.dart     ← The card widget for one entry
└── utils/                  ← Helper functions (you'll add storage.dart here in Stage 3)
```
