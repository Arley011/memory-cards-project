# Memory Cards — Flutter Course

Welcome! In this course you will build **Memory Cards**, a personal journaling app using Flutter.

Each entry in the app has a **title**, a **short text**, and a **date**. Over the week you'll add navigation, data persistence, categories, and your own feature of choice.

---

## Quick start

1. **Install Flutter + VS Code** → [docs/setup.md](docs/setup.md)
2. **Clone or fork this repository**
3. Open the `starter/` folder in VS Code
4. Run `flutter pub get` in the terminal
5. Connect a device or start an emulator, then `flutter run`
